## Custom Select

Replace the browser's native `<select>` with a fully styled, searchable, keyboard-navigable dropdown.

### Variants

1. **Searchable single select** — type to filter options
2. **Option groups** — options organized under group labels
3. **Multi-select** — select multiple, displays as removable tag chips

### Keyboard support

`↑ ↓` navigate options · `Enter` selects · `Escape` closes · `Tab` closes and moves on
