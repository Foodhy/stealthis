## React Native Button Variants

A complete button system for React Native apps — four visual variants, three sizes, loading state, disabled state, and optional haptic feedback via Expo.

### Variants

- **Solid** — filled background, white text
- **Outline** — transparent with colored border and text
- **Ghost** — no border, subtle press highlight
- **Icon** — circular button with centered icon

### Sizes

| Size | Height | Font | Padding |
|---|---|---|---|
| `sm` | 32 | 13 | 12h |
| `md` | 44 | 15 | 20h |
| `lg` | 52 | 17 | 28h |

### Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `variant` | `'solid' \| 'outline' \| 'ghost' \| 'icon'` | `'solid'` | Visual style |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Button size |
| `color` | `string` | `'#6366f1'` | Accent color |
| `loading` | `boolean` | `false` | Shows spinner, disables press |
| `disabled` | `boolean` | `false` | Reduces opacity, disables press |
| `haptic` | `boolean` | `false` | Trigger light haptic on press (Expo only) |
| `onPress` | `() => void` | — | Press handler |

### Usage

```tsx
<Button variant="solid" size="md" onPress={() => alert('Pressed!')}>
  Save Changes
</Button>

<Button variant="outline" color="#ef4444" loading>
  Deleting…
</Button>

<Button variant="icon" size="lg" haptic onPress={toggleFav}>
  <HeartIcon />
</Button>
```

### How it works

Built on `Pressable` for full control over press states. Uses `Animated.Value` to drive opacity on press-in/press-out. The `loading` state swaps children for an `ActivityIndicator` and disables interaction. Haptic feedback uses `expo-haptics` when available (graceful no-op on bare RN).
