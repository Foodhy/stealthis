Next.js + Prisma + PostgreSQL

Full-stack starter with typed DB access & migrations

Quick Start
1 npx create-next-app@latest my-app --ts
2 npm i prisma @prisma/client
3 npx prisma init
Generated Structure
prisma/
  schema.prisma — data model & DB connection
  migrations/ — version-controlled SQL
src/
  lib/
    prisma.ts — singleton client instance
  app/
    api/
      route.ts — typed API handlers
.env — DATABASE_URL
Key Files
prisma/schema.prisma
Declarative data model. Define your tables, relations, and enums. Prisma generates types and SQL migrations from this file.
src/lib/prisma.ts
Singleton pattern using globalThis to prevent connection pool exhaustion during Next.js hot reloads in development.
src/app/api/*/route.ts
App Router API routes. Import the singleton client and get full autocompletion for all queries, including relations.
Setup Flow
Install → Init → Define Schema → Migrate → Seed → Use Client
