## Expo Barcode Scanner

A polished barcode and QR code scanner interface built for React Native and Expo. This implementation simulates the full scanning experience — animated scan line, result detection, history tracking, and action sheets — all without requiring camera hardware access.

### Features

- **Scanner Overlay** — Dark corner brackets framing a transparent scanning area, replicating the look of production barcode scanners.
- **Animated Scan Line** — A glowing line sweeps vertically through the scanning region on a continuous loop.
- **Simulated Scanning** — Tap the scan area to trigger a simulated barcode detection, generating random QR codes, EAN-13, or Code128 results.
- **Result Modal** — Displays the scanned barcode type, decoded data, and timestamp in a styled action sheet.
- **Action Buttons** — Copy to clipboard, open URL (when a URL is detected), and share scanned data.
- **Scan History** — A scrollable list of all previous scans displayed below the scanner viewport.
- **Torch Toggle** — Simulated flashlight button for the authentic scanner experience.

### Usage

Mount the component and tap the scanning area to simulate a scan. Results appear in a modal with contextual action buttons. All scans are automatically logged to the history list.

### Customization

- Adjust the scan area dimensions by modifying the overlay layout constants.
- Change the scan line color and animation speed for different visual effects.
- Extend the action buttons with custom handlers for deep linking or API lookups.
