## Event / Conference Landing

A complete conference landing page with all the sections a real event needs: countdown, speakers, schedule, sponsors, and ticket purchase CTA.

### Sections

1. **Hero** — Event name, date/location, live countdown timer
2. **Speakers** — Card grid with photo, name, title, company
3. **Schedule** — Day-by-day timeline with talk titles and times
4. **Sponsors** — Three-tier logo marquee (Gold / Silver / Community)
5. **Tickets** — Pricing table with tier comparison
6. **Footer** — Newsletter signup + social links

### Key interactions

- **Countdown timer** — real-time `setInterval` updating days/hours/minutes/seconds
- **Speaker cards** — hover reveals LinkedIn + talk topic
- **Schedule tabs** — animated day switcher
- **Sponsor marquee** — CSS infinite scroll, pauses on hover
- Lenis smooth scroll throughout
