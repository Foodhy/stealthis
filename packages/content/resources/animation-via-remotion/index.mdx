A Remotion starter that registers **8 compositions** in a single `Root.tsx`. Each composition demonstrates a core animation technique.

## Included compositions

1. Typewriter
2. Spring bounce
3. Bar chart
4. Slideshow (Series)
5. Counter
6. Easing curves
7. Pie chart (SVG donut)
8. Skills presentation (multi-scene)

## How to use

- The Remotion project lives in `apps/remotion`.
- Compositions are registered in `apps/remotion/src/Root.tsx`.
- Add new animations under `apps/remotion/src/animations/`.

## Remotion project

The live Remotion project for these compositions lives inside this repo under `apps/remotion`.
