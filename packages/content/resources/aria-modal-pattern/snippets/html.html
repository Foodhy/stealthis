Accessible modal dialogs with focus trap, Escape to close, and screen reader announcements.

Alert Dialog

Form Dialog

Modal with form fields, validation, and submit handling.

Accessibility Features

  • Focus trap: Tab and Shift+Tab cycle within the modal
  • Escape closes the modal and returns focus to trigger
  • Background scroll is locked while modal is open
  • role="dialog" + aria-modal="true" for screen readers
  • aria-labelledby and aria-describedby provide context
