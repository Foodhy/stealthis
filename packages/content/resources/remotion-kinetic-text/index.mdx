## Kinetic Typography

Kinetic typography with staggered spring-animated words entering from below. Each word springs up with configurable damping and stiffness, scaling from 0.6 to full size. The last word is highlighted in an accent color for emphasis. After the entrance sequence, words exit upward with a fade — each word staggered slightly for a smooth wave-out effect.

### Composition specs

| Property | Value |
|---|---|
| Resolution | 1280 x 720 |
| FPS | 30 |
| Duration | 5 s (150 frames) |

### Customization

- **WORDS** — array of strings displayed in sequence
- **ACCENT** — highlight color applied to the last word (default `#f59e0b`)
- **STAGGER** — number of frames between each word entrance (default `8`)
