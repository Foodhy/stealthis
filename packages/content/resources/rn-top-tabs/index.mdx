## Overview

A lightweight, dependency-free top tab bar for React Native that supports horizontal scrolling when there are many tabs. The active tab is highlighted with a smoothly animated underline indicator powered by `Animated.spring`, and panels can be switched by swiping horizontally thanks to a paging `ScrollView`.

## Features

- **Scrollable tab bar** — handles any number of tabs without overflow by wrapping them in a horizontal `ScrollView`.
- **Animated underline** — a spring-driven indicator slides and resizes to match the active tab's position and width.
- **Swipe navigation** — panels sit inside a horizontal, paging-enabled `ScrollView` for native-feeling gesture switching.
- **Active state styling** — the selected tab label turns white while inactive labels stay gray.
- **Zero external dependencies** — built entirely with `react` and `react-native` core APIs.

## Usage

Pass an array of tab objects, each with a unique `key`, a display `label`, and a `content` node to render inside that tab's panel.

```tsx
<TopTabs
  tabs={[
    { key: "home", label: "Home", content: <HomeScreen /> },
    { key: "settings", label: "Settings", content: <SettingsScreen /> },
  ]}
/>
```

## Customisation ideas

- Swap the underline for a pill-shaped background highlight.
- Add icons next to (or instead of) text labels.
- Replace the `ScrollView` panels with a `FlatList` for lazy rendering of heavy content.
