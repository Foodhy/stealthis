ARIA Combobox

Accessible autocomplete implementing ARIA 1.2 combobox pattern with keyboard navigation.

Selected: None

ARIA Attributes

role="combobox" Container wrapping the input
role="listbox" The dropdown options list
role="option" Each selectable item
aria-expanded Whether the listbox is open
aria-activedescendant ID of the focused option
aria-autocomplete="list" Filtering suggestions are shown

Keyboard Controls

Down Open listbox / next option Up Previous option Enter Select highlighted option Escape Close listbox Home First option End Last option
