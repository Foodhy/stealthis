Workflow Structure
.github/ workflows/ ├── ci.yml (Lint + Test + Build) ├── deploy-staging.yml ├── deploy-production.yml ├── release.yml (Semantic release) └── reusable/ ├── test.yml (workflow_call) └── build.yml (workflow_call) actions/ └── setup/ (Composite action) └── action.yml CODEOWNERS

PR → Lint → Test (matrix) → Build → Staging → Approval → Production

Environments

Staging

Production

Required reviewers (2). Wait timer: 5 min. Branch protection: main only. Rollback on failure.
