## Architecture Firm

An award-worthy architecture firm website. Full-bleed imagery, horizontal scroll project gallery, editorial typography, and the restraint of a firm that lets the work speak.

### Sections

1. **Nav** — Minimal wordmark left, links right, thin bottom border
2. **Hero** — Full-viewport image with overlaid studio name + year founded
3. **Philosophy** — Large serif quote, small descriptor text
4. **Projects** — Horizontal pinned scroll gallery (GSAP ScrollTrigger horizontal)
5. **About** — Split layout: text left, image right, counter stats
6. **Awards** — Clean list with year, award name, project
7. **Contact** — Single-field "Tell us about your project" minimal form

### Key interactions

- Horizontal project gallery pinned with GSAP `scrub`
- Image reveal on scroll via `clip-path` wipe
- Cursor that enlarges on hover over project images ("View")
- Lenis smooth scroll for the editorial pacing feel
