Roving Tabindex

Single Tab stop per widget group — arrow keys navigate within, Tab moves between groups

How it works

Only one element per group has tabindex="0"; all others have tabindex="-1". Pressing an arrow key moves tabindex="0" to the next item and focuses it. Tab moves to the next group, not the next item.
