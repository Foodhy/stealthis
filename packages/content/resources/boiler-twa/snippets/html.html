TWA — Trusted Web Activity

Wrap any PWA as an Android app and publish to Google Play Store. No Java/Kotlin required.

Prerequisites
  • PWA with a valid web app manifest
  • HTTPS hosting
  • Android SDK installed (or let Bubblewrap download it)
Bubblewrap CLI Commands
Initialize from your PWA manifest:
bubblewrap init --manifest=https://your-pwa.com/manifest.webmanifest
Build the signed APK/AAB:
bubblewrap build
Generated Project

build.gradle

Setup Flow
PWA → bubblewrap init → Build APK → Asset Links → Play Store
Digital Asset Links
Required: Place assetlinks.json at /.well-known/assetlinks.json on your domain to prove ownership and hide browser chrome.
// .well-known/assetlinks.json [{ "relation": ["delegate_permission/common.handle_all_urls"], "target": { "namespace": "android_app", "package_name": "com.your.package", "sha256_cert_fingerprints": ["AA:BB:CC:..."] } }]
