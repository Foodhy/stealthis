## Expo Biometric Authentication

A biometric authentication component for React Native apps built with Expo. It leverages `expo-local-authentication` to provide Face ID, Touch ID, and fingerprint authentication with graceful fallback to device passcode.

### Features

- **Hardware detection** — checks whether the device supports biometric authentication before prompting.
- **Enrollment check** — verifies that the user has enrolled biometrics (fingerprint, face, iris) on the device.
- **Supported types** — queries which authentication methods are available (fingerprint, facial recognition, iris).
- **Authentication flow** — presents a lock screen UI with an animated lock icon that transitions to unlocked on success.
- **Fallback to passcode** — if biometric auth is unavailable or fails, the user can authenticate with their device passcode.
- **Retry on failure** — displays a retry button so users can re-attempt authentication after a failed attempt.

### Usage

Import the component and render it as a gate before protected content. The `onAuthenticated` callback fires when the user successfully authenticates.

```tsx
import BiometricAuth from "./BiometricAuth";

<BiometricAuth onAuthenticated={() => navigateToHome()} />;
```

### Dependencies

- `expo-local-authentication`
