## React Native Section List

A polished, grouped section list component for React Native featuring sticky headers with item count badges, an alphabet index sidebar for fast scrolling, and customizable separators. Ideal for contacts, directories, or any alphabetically-grouped data.

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `sections` | `SectionData[]` | required | Array of section objects with `title` and `data` fields. |
| `renderItem` | `(info: { item, index, section }) => ReactElement` | Built-in row | Custom renderer for each list item. |
| `renderSectionHeader` | `(info: { section }) => ReactElement` | Built-in sticky header | Custom renderer for section headers. |
| `showAlphabetIndex` | `boolean` | `true` | Whether to display the vertical alphabet sidebar on the right edge. |
| `separatorColor` | `string` | `"rgba(255,255,255,0.08)"` | Color of the line separators between items. |
| `backgroundColor` | `string` | `"#0f172a"` | Background color of the list container. |

### Usage

```tsx
import SectionListScreen from "./SectionListScreen";

// Drop it in as your root component or a screen in your navigator:
export default function App() {
  return <SectionListScreen />;
}
```

You can also use the `GroupedSectionList` component directly with your own data:

```tsx
import { GroupedSectionList } from "./SectionListScreen";

const myData = [
  { title: "A", data: [{ id: "1", name: "Alice", subtitle: "Engineer" }] },
  { title: "B", data: [{ id: "2", name: "Bob", subtitle: "Designer" }] },
];

<GroupedSectionList sections={myData} showAlphabetIndex />;
```

### How it works

1. **SectionList** from React Native is used with `stickySectionHeadersEnabled` so that each letter header stays pinned at the top while scrolling through its section.
2. Each **section header** displays the letter and an item count badge, styled with a semi-transparent background that blurs into the dark theme.
3. **Item rows** render an avatar circle derived from the contact's initials, alongside the name and subtitle. Rows are separated by subtle divider lines whose color is configurable.
4. The **alphabet index** is an absolutely-positioned vertical strip of tappable letters on the right edge of the screen. Tapping a letter calls `scrollToLocation` on the SectionList ref to jump directly to that section, enabling quick navigation through long lists.
5. The entire component is self-contained with no external dependencies beyond `react` and `react-native`.
