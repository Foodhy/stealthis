## Parallax Scroll

A parallax scrolling system for React Native that moves layered elements at different speeds as the user scrolls, creating an illusion of depth and visual richness.

### How it works

`ParallaxScrollView` wraps an `Animated.ScrollView` and tracks the vertical scroll offset via `Animated.event`. Child `ParallaxLayer` components accept a `speed` prop that controls how fast they move relative to the scroll:

- `speed: 0` — fixed in place (no movement)
- `speed: 0.5` — moves at half the scroll speed (background feel)
- `speed: 1` — moves at normal scroll speed
- `speed: 2` — moves faster than scroll (foreground feel)

Each layer uses `interpolate` to map the scroll offset to a `translateY` value scaled by the speed factor.

### Features

- Configurable per-layer speed for fine-grained depth control
- `ParallaxImage` convenience component for background images that drift behind content
- Hero section support with overlapping text and parallax background
- Smooth 60fps performance using native Animated driver

### When to use it

- Landing pages with hero imagery
- Storytelling or editorial content with visual depth
- Product showcases with layered card sections
- Any scroll-heavy screen that benefits from a sense of dimension
