Box Reveal Effect

A dramatic two-step animation where a colored block sweeps across the element, revealing polished content underneath. Perfect for hero headlines and feature callouts.

Custom Colors

Change the reveal color with a simple CSS variable. Match your brand, create emphasis, or use different colors for different sections.

Fast

Pure CSS animations with no JavaScript overhead during the transition phase.

Flexible

Works with any content — text, cards, images, or entire sections.

Staggered Reveals

Cascading Wipe

Elements within the same group automatically stagger their reveals, creating a beautiful sequential wipe effect down the page.
