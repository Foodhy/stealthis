## Expo File System

A comprehensive file system component for React Native apps built with Expo. It uses `expo-file-system` to download files with progress tracking, read and write local files, and browse the document directory.

### Features

- **Download with progress** — uses `createDownloadResumable` to download files from a URL while displaying a real-time progress bar.
- **File info** — retrieves file metadata (size, existence, modification time) via `getInfoAsync`.
- **Read/write text files** — write arbitrary text to local storage with `writeAsStringAsync` and read it back with `readAsStringAsync`.
- **Directory listing** — browse the contents of the document directory using `readDirectoryAsync`.
- **Sections UI** — organized into three sections: download, file browser, and text editor.

### Usage

Render the component to get a full file management UI. All operations target the app's document directory.

```tsx
import FileSystemDemo from "./FileSystemDemo";

<FileSystemDemo />;
```

### Dependencies

- `expo-file-system`
