Route Groups — Folder Structure
app/
  (auth)/
    login/page.tsx # /login
    register/page.tsx # /register
    layout.tsx # Centered card, no sidebar
  (dashboard)/
    page.tsx # / (dashboard home)
    analytics/page.tsx # /analytics
    users/page.tsx # /users
    settings/page.tsx # /settings
    layout.tsx # Sidebar + header layout
  components/
    ui/ # shadcn/ui primitives
  layout.tsx # Root layout (providers, fonts)
Key Patterns
  • Server ComponentsFetch data on the server — no client-side waterfalls
  • Route GroupsSeparate auth/dashboard layouts without affecting URLs
  • Parallel RoutesModal overlays via @modal slot convention
  • MiddlewareAuth redirects before any rendering occurs
  • shadcn/uiConsistent, accessible component primitives
