## Dropdown Menu

An accessible action menu that opens on button click with full keyboard navigation.

### Features

- Arrow keys navigate items
- `Enter` / `Space` activates item
- `Escape` closes menu
- Closes on outside click
- Dividers and disabled items
- Destructive (red) item variant
- Icon support per item

### Demo variants

1. **Action menu** — Edit / Duplicate / Delete with icons
2. **Account menu** — avatar trigger with Profile / Settings / Sign out
3. **Select-like** — checkmark on selected option
