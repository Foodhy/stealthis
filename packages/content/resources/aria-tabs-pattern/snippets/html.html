ARIA Attributes Used

  • role="tablist" — Container for the tab buttons
  • role="tab" — Each clickable tab button
  • role="tabpanel" — Content panel associated with a tab
  • aria-selected="true/false" — Indicates the active tab
  • aria-controls — Links a tab to its panel
  • aria-labelledby — Links a panel back to its tab
  • tabindex="0/-1" — Roving tabindex for keyboard focus

Keyboard Controls

Left / Right — Move between tabs Home — Go to first tab End — Go to last tab Tab — Enter/leave the tab panel
