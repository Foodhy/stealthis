## Code Walkthrough

A code walkthrough that types out code line-by-line in a dark editor frame with a title bar (macOS-style traffic lights), line numbers, and syntax coloring. The editor window fades and scales in, then the typing cursor advances through each line at a configurable speed. Each line uses its first color token for simplified syntax highlighting.

### Composition specs

| Property | Value |
|---|---|
| Resolution | 1280 x 720 |
| FPS | 30 |
| Duration | 8 s (240 frames) |

### Customization

- **CODE_LINES** — array of `{ text, colors }` objects defining each line and its syntax colors
- **CHARS_PER_FRAME** — typing speed in characters per frame (default `1.5`)
- **TITLE** — filename shown in the editor title bar (default `server.ts`)
