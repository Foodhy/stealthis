## Redline Portfolio

A minimal, high-contrast portfolio design featuring a strict black and red color palette. Built with focus on UI systems, capabilities, and clean data presentation.

### How it works

The portfolio uses a data-driven approach with dynamic content generation:

1. **Data Structure** — Projects, capabilities, stack, and stats defined in JavaScript
2. **GSAP Animations** — Smooth fade-in animations for cards and content
3. **Lenis Smooth Scroll** — Enhanced scrolling experience
4. **Dark Mode Toggle** — Interactive button to switch between light and dark backgrounds
5. **Grid Layouts** — Responsive grid systems for projects and contact cards

### Key features

- High-contrast red and black design
- Data-driven content generation
- Smooth GSAP animations
- Dark mode toggle functionality
- Responsive grid layouts
- Minimal, focused UI
- Clean typography with Oswald and Manrope fonts

### When to use it

- UI/UX designer portfolios
- Minimal portfolio designs
- High-contrast interfaces
- Data-focused presentations
- Creative agency showcases
