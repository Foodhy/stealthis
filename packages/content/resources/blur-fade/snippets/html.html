Smooth Reveal

Content emerges from a soft blur as it enters the viewport, creating a polished and modern scroll experience. Each element transitions independently with staggered timing.

◈

Performance

Hardware-accelerated CSS transitions keep animations silky smooth at 60fps.

◎

Accessible

Uses IntersectionObserver for efficient, non-blocking scroll detection.

✦

Customizable

Control blur amount, duration, easing, and stagger timing via CSS variables.

Cascading Effect

Elements within the same parent container automatically receive staggered delays, creating a beautiful cascading reveal without any extra configuration.

★

Zero Dependencies

Pure CSS and vanilla JavaScript. No libraries required.

⚙

Easy Setup

Just add the blur-fade class to any element you want to reveal.

✨

One-time Trigger

Elements are unobserved after reveal, keeping the observer lightweight.
