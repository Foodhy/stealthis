## Search Results Page

A Google-style search results page with a search bar, filter sidebar, result cards showing title, URL, and description with highlighted keywords, sorting, and pagination.

### Features

- **Search bar** — pre-filled query with re-search capability
- **Filter sidebar** — Type (All, Articles, Docs, Videos), Date (Any time, Past week, Past month, Past year), Sort by (Relevance, Date, Popularity)
- **Result count** — "Showing 1-10 of 247 results for 'design systems'"
- **Result cards** — title link, breadcrumb URL, description with keyword highlights
- **Pagination** — numbered pages with prev/next
- **Empty state** — no results found suggestions

### When to use it

- Site-wide search results
- Documentation search
- Knowledge base search
