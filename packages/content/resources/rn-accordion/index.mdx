## React Native Accordion

A fully animated accordion component built with the React Native Animated API. Each item smoothly expands and collapses with a measured content height transition, and a chevron indicator rotates to signal the current state. Supports both single-expand (only one item open at a time) and multi-expand modes.

### Props

#### `Accordion`

| Prop            | Type        | Default | Description                                      |
|-----------------|-------------|---------|--------------------------------------------------|
| `allowMultiple` | `boolean`   | `false` | When false, opening one item closes the others.  |
| `children`      | `ReactNode` | —       | One or more `AccordionItem` children.            |

#### `AccordionItem`

| Prop          | Type        | Default | Description                                 |
|---------------|-------------|---------|---------------------------------------------|
| `title`       | `string`    | —       | Header text displayed on the pressable bar. |
| `children`    | `ReactNode` | —       | Content revealed when the item is expanded. |
| `defaultOpen` | `boolean`   | `false` | Whether the item starts in the open state.  |

### Usage

```tsx
<Accordion allowMultiple={false}>
  <AccordionItem title="What is React Native?">
    <Text>A framework for building native apps using React.</Text>
  </AccordionItem>
  <AccordionItem title="Is it cross-platform?" defaultOpen>
    <Text>Yes — iOS, Android, and more.</Text>
  </AccordionItem>
</Accordion>
```

### How it works

1. Each `AccordionItem` renders its children inside a hidden measuring container to capture the full content height via `onLayout`.
2. When toggled, the component animates from height `0` to the measured height (or vice-versa) using `Animated.timing` with a 300 ms duration.
3. The chevron arrow uses an interpolated rotation from `0deg` to `90deg` synced to the same animated value.
4. In single-expand mode, the parent `Accordion` tracks the currently open index and collapses siblings automatically.
