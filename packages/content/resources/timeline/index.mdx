## Timeline

A vertical timeline component with date markers, status indicators, and content cards. Supports both single-column and alternating left/right layouts.

### Variants

- **Single column** — events stack left-aligned, ideal for sidebars
- **Alternating** — events alternate left/right of the center spine for a classic timeline look

### States

- `past` — completed events with a filled accent dot
- `present` — current event highlighted with a glowing pulse ring
- `future` — upcoming events with a dimmed unfilled dot

### Features

- Pure CSS — no JavaScript required
- Responsive (collapses to single column on mobile)
- `prefers-reduced-motion` support for the pulse animation
- Accessible semantic markup with `<time>` elements
