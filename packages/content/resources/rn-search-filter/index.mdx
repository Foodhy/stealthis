## Search & Filter List

A production-ready searchable list component for React Native. Type into the search bar and results filter in real-time with a 300ms debounce for smooth performance. Matching text is highlighted directly in each result row.

### Features

- **Debounced input** — filters after 300ms of inactivity to avoid excessive re-renders
- **Highlighted matches** — matching substrings are rendered in bold with an accent color
- **Recent searches** — when the input is focused but empty, previously searched terms appear as quick-access chips
- **Empty state** — a friendly message displays when no results match the current query
- **Customizable** — pass your own `renderItem`, choose which `searchKeys` to match against, and set a custom placeholder

### Usage

Provide an array of data objects and specify which keys to search. The component handles input, filtering, and rendering internally. Wrap your own item UI via `renderItem` for full control over how each row looks.
