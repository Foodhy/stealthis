## Overview

A collapsible header that responds to scroll events with buttery-smooth animations. As the user scrolls up, the large hero image compresses with a parallax effect, the title fades out, and a compact sticky header bar fades in — all driven by a single `Animated.Value` and `interpolate()` with the native driver.

## Features

- **Parallax hero image** — the background image translates at half the scroll speed, creating a natural depth effect.
- **Fading title** — the large title gradually disappears as the header collapses.
- **Sticky compact header** — a slim bar with the title fades in once the large header is fully collapsed.
- **Native driver** — all animations run on the native thread via `useNativeDriver: true` for 60 fps performance.
- **Configurable height** — pass `headerHeight` to control the expanded header size.
- **Zero external dependencies** — uses only `react` and `react-native` built-ins.

## Usage

Wrap your scrollable content inside the `AnimatedHeader` component and provide a title and background image URI.

```tsx
<AnimatedHeader
  title="Explore"
  imageUri="https://picsum.photos/800/400"
  headerHeight={220}
>
  {/* Your content here */}
</AnimatedHeader>
```

## Customisation ideas

- Add a gradient overlay on the hero image for better title readability.
- Include action buttons (search, share) in the compact header bar.
- Support a subtitle or tag line that fades independently of the main title.
- Add a snap-to-expanded / snap-to-collapsed behaviour using `Animated.event` with `onScrollEndDrag`.
