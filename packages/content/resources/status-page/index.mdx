## Status Page

A public-facing system status page showing service health, uptime history, and incident reports. Inspired by Statuspage.io and Instatus.

### Features

- **Overall status banner** — All Systems Operational (green) or degraded/outage states
- **Service list** — each service with name, status indicator (operational/degraded/outage), uptime %
- **Uptime bars** — 90-day history bars showing daily status (green/yellow/red)
- **Incident timeline** — recent incidents with severity, description, updates, timestamps
- **Subscribe** — email subscribe for status updates
- **Auto-refresh** — simulated live status updates

### When to use it

- Public status page for SaaS products
- API health monitoring dashboard
- Infrastructure status display
