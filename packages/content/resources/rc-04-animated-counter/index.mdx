## useAnimatedCounter Hook

Animate any number when it scrolls into view. Returns a ref to attach to the container and the current display value as a string.

### Usage

```tsx
const { ref, display } = useAnimatedCounter({
  target: 12500,
  prefix: "$",
  suffix: "+",
  duration: 1800,
});

<div ref={ref}>{display}</div>
```

### Options

| Prop | Default | Description |
|---|---|---|
| `target` | — | The number to count up to |
| `duration` | `1800` | Animation duration in ms |
| `prefix` | `""` | String prepended to the value (e.g. `"$"`) |
| `suffix` | `""` | String appended to the value (e.g. `"+"`, `"%"`) |
| `decimals` | `0` | Decimal places to display |
| `easing` | `"easeOut"` | `"easeOut"` \| `"easeInOut"` \| `"linear"` |

### How it works

1. An `IntersectionObserver` watches the returned ref
2. When the element enters the viewport, a `requestAnimationFrame` loop runs the easing function over `duration` ms
3. Each frame, the current value is formatted with `toLocaleString()` + prefix/suffix and stored in state

Respects `prefers-reduced-motion` — immediately shows the final value without animation.
