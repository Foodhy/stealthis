## Site Footer

A minimal, dark-themed footer with a two-column layout: brand section on the left and grouped navigation links on the right. Sits at the bottom of any page without adding visual weight.

### Features

- CSS Grid two-column layout
- Grouped navigation links with uppercase labels
- Responsive: stacks to single column on mobile
- Bottom divider with copyright line

### When to use

- Marketing or landing pages
- SaaS product sites
- Any project needing a clean footer row
