## Expo Map View

A fully interactive map component built for React Native and Expo. This implementation simulates a map view with coordinate-based marker positioning, pan gestures for viewport navigation, zoom controls, and tappable marker callouts — all without external mapping dependencies.

### Features

- **Custom Markers** — Position markers by latitude and longitude, rendered as colored pins on the map surface with tap-to-reveal callout bubbles.
- **Pan Gesture Navigation** — Drag the map to pan across the viewport using React Native's PanResponder.
- **Zoom Controls** — Dedicated zoom in/out buttons to adjust the visible region.
- **User Location Indicator** — Animated pulsing blue dot representing the current device location.
- **Search Bar** — Filter or search for points of interest directly on the map.
- **Region Change Callback** — `onRegionChange` prop fires whenever the viewport moves, enabling parent components to react to navigation.

### Usage

Pass an `initialRegion` with latitude, longitude, and deltas, along with an array of `markers` each containing coordinates, a title, and a description. Enable `showUserLocation` to display the pulsing location dot.

### Customization

- Adjust marker colors individually via the `color` property on each marker object.
- Swap the coordinate grid style for satellite-like imagery by changing the background and grid colors.
- Extend callout content with images, ratings, or action buttons.
