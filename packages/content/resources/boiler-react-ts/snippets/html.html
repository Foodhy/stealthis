R

React + TypeScript + Vite

Production-ready starter with ESM dev server & Rollup builds

Quick Start

$ npm create vite@latest my-app -- --template react-ts
$ cd my-app
$ npm install
$ npm run dev # http://localhost:5173

Key Files

vite.config.ts Plugins, aliases, build options
tsconfig.json Project references for app & node configs
main.tsx Entry point, renders into #root via createRoot
index.html Vite entry with module script to main.tsx
