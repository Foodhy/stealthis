Keyboard Guide

← → Move between menubar items
↓ ↑ Navigate within a dropdown
Enter / Space Open dropdown or activate item
Escape Close dropdown, return to menubar
Home / End Jump to first / last item
A–Z Type-ahead: jump to matching item
