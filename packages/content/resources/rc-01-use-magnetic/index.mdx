## useMagnetic Hook

A generic React hook that turns any element into a magnetic attractor — it pulls toward the cursor when it gets close, then snaps back smoothly.

### Usage

```tsx
const { ref, onMouseMove, onMouseLeave } = useMagnetic({ strength: 0.4, radius: 120 });

<button ref={ref} onMouseMove={onMouseMove} onMouseLeave={onMouseLeave}>
  Click me
</button>
```

### Options

| Prop | Default | Description |
|---|---|---|
| `strength` | `0.4` | Pull intensity (0 = none, 1 = full cursor offset) |
| `radius` | `120` | Distance in px at which the effect activates |

### How it works

1. On `mousemove`, compute distance from cursor to element center
2. If within `radius`, apply a `translate` transform proportional to the offset × `strength`
3. On `mouseleave`, clear the transform (CSS transition handles the snap-back)

Respects `prefers-reduced-motion` — effect is disabled when the user opts out.
