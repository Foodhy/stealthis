## Wishlist Page

A saved items / wishlist page for e-commerce. Shows favorited products with options to move to cart, remove, or share the entire wishlist.

### Features

- **Product list** — cards with image, title, price, stock status, date added
- **Actions** — Move to Cart button, Remove (heart toggle) button per item
- **Share wishlist** — copy link or share buttons
- **Sort by** — Date Added, Price, Name
- **Empty state** — "Your wishlist is empty" with CTA to browse products
- **Item count** — "6 saved items"

### When to use it

- E-commerce wishlist page
- Saved items / bookmarks page
- Favorite products list
