Real-time WCAG contrast ratio checker that calculates relative luminance and contrast ratio between any two colors. Evaluates against AA (4.5:1 normal, 3:1 large) and AAA (7:1 normal, 4.5:1 large) standards with instant visual feedback and preset color pairs.
