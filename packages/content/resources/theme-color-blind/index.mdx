Simulates color vision deficiencies (protanopia, deuteranopia, tritanopia) using SVG color matrix filters applied in real-time. Includes a dashboard-like UI with colored status indicators and charts, plus an accessible alternative that uses patterns, icons, and text labels instead of relying on color alone.
