A reusable prompt to analyze and adapt system prompts from public AI tool repositories.

## What This Prompt Solves

- Extracts structure, tone, constraints, and output contracts from source prompts.
- Adapts reference prompts to your use case without blindly copying behavior.
- Produces a documented adaptation with clear source links.

## Recommended Inputs

- Source prompt text and repository URL.
- Target use case, constraints, and risk boundaries.
- Desired output style and quality criteria.

## Expected Output

- Adapted prompt draft.
- Diff notes vs source intent.
- Risk/limitation notes and validation checklist.
