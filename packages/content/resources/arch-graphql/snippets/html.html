Schema-first vs Code-first

Schema-first (SDL)

Write .graphql files defining types, then generate TypeScript types. Clear contract, great for team collaboration.

Tools: Apollo Server, GraphQL Yoga, graphql-tools

Code-first (Programmatic)

Define schema using TypeScript decorators or builder APIs. Single source of truth with full IDE support and no codegen step.

Tools: TypeGraphQL, Nexus, Pothos
Request Flow
Query → Parse → Validate → Resolver → DataSource → Response

N+1 Problem & DataLoader

Fetching 50 users with posts = 1 + 50 queries without batching. DataLoader batches and deduplicates per-request: collects all IDs in a single tick, fires one batched query, then distributes results back to each resolver.
