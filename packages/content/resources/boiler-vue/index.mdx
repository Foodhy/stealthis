## Vue 3 + Vite Starter

The official `create-vue` scaffolding tool builds a Vue 3 project powered by Vite with an interactive CLI that lets you pick exactly the features you need: TypeScript, Vue Router, Pinia, Vitest, ESLint, and Prettier.

### Quick Start

```bash
npm create vue@latest my-app
cd my-app
npm install
npm run dev
```

The interactive prompts let you toggle TypeScript, JSX, Vue Router, Pinia, Vitest, E2E testing (Cypress/Playwright), ESLint, and Prettier.

### What You Get

- **Vue 3 Composition API** вЂ” `<script setup>` with reactive refs, computed properties, and composables
- **Vite** вЂ” Instant dev server, native ESM, optimized production builds
- **TypeScript** вЂ” Full type inference for props, emits, slots, and the template
- **Vue Router** вЂ” File-based mental model with nested routes, guards, and lazy loading
- **Pinia** вЂ” Lightweight, type-safe state management replacing Vuex
- **Vitest** вЂ” Vite-native testing (optional)
- **ESLint + Prettier** вЂ” Code quality and formatting (optional)

### Key Files

- **`vite.config.ts`** вЂ” Vue plugin, path aliases (`@/` maps to `src/`), dev server options.
- **`src/router/index.ts`** вЂ” Route definitions with `createRouter` and `createWebHistory`.
- **`src/stores/counter.ts`** вЂ” Example Pinia store using `defineStore` with Composition API syntax.
- **`src/App.vue`** вЂ” Root component with `<RouterView />` for page rendering.
