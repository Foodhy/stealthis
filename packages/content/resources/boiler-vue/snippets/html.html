Vue 3 + Vite

Composition API, Vue Router, Pinia & TypeScript

Quick Start

$ npm create vue@latest my-app
# Interactive prompts: TS, Router, Pinia, Vitest, ESLint...
$ cd my-app
$ npm install
$ npm run dev # http://localhost:5173
